# Query Prompt

```text
위키에서 다음 질문에 대한 답을 찾아줘(query).

먼저 architecture.md로 관련 후보를 좁혀줘. 필요한 경우 AI-Sessions/wiki/maps, AI-Sessions/wiki/research, AI-Sessions/wiki/harness 안의 해당 노트만 읽어서 질문에 직접 답해줘.

논문·구현·아이디어에 관한 질문이면 세 레이어를 함께 읽어라:
- `research/papers/<slug>.md` (논문 내용·abstract)
- `research/sources/<slug>-code.md` (구현 분석·URL·패턴)
- `research/ideas/idea-*.md` (연결된 아이디어·가설)

이건 작업 재개(reference)가 아니라 단일 사실 확인이니, 새 문서를 만들지 말고 읽기 전용으로 간결하게 답해줘. 근거가 된 노트는 함께 알려줘.
```
