# Ingest Prompt

```text
AI-Sessions/raw/에 추가된 자료를 ingest 해줘.

raw 원본은 절대 수정하지 말고, 자료 종류에 맞게 wiki 문서를 만들거나 기존 노트를 갱신해줘. 새로 만들기 전에 index.md로 관련 노트를 먼저 찾고, 같은 주제가 있으면 갱신하고, 상충하는 내용은 덮어쓰지 말고 표시해줘.

- 논문이면: wiki/papers/에 인사이트 노트(문제·핵심 아이디어·방법·한계·내 연구와의 연결)를 만들고, 반복 개념은 wiki/concepts/로 분리해줘.
- 레퍼런스 구현 GitHub 레포면: wiki/sources/에 구현 분석 노트(구현하는 방법·아키텍처·핵심 파일·실행법·내 구현에 가져올 패턴)를 만들고, 대응되는 논문 노트가 있으면 [[링크]]로 연결해줘.
- 아이디어·인사이트면: wiki/ideas/에 저장하고 근거가 된 논문/레포 노트로 [[링크]]해줘.

의사결정 수준의 내용은 wiki/decisions/에, 구현 작업은 wiki/dev-tasks/에 연결해줘.

작업이 끝나면 index.md와 log.md를 갱신해줘.
```
