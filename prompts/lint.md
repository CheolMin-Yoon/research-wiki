# Lint Prompt

```text
이 Obsidian 연구 위키를 lint 해줘.

다음을 점검해줘.

1. raw 원본이 수정되었거나 wiki 영역에 섞여 있지 않은가?
2. 중요한 wiki 문서가 index.md에 누락되어 있지 않은가?
3. 중요한 작업이 log.md에 기록되어 있지 않은가?
4. 저장의 5가지 필터를 통과하지 못한 일회성 정보가 wiki에 저장되어 있지 않은가?
5. papers와 sources 노트 사이에 끊어진 [[링크]]나 고아 문서가 없는가?
6. 민감정보, API 키, 토큰, 개인정보가 저장되어 있지 않은가?
7. CLAUDE.md와 AGENTS.md 규칙이 서로 어긋나지 않는가?

문제가 있으면 파일별로 수정 제안을 정리해줘.
```
